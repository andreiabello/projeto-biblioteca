# 📚 Projeto Biblioteca

Projeto de análise de empréstimos de livros utilizando SQL, Excel e Tableau.

## Objetivo
Analisar dados de empréstimos de uma biblioteca para entender:
- Quantidade de livros emprestados
- Autores mais lidos
- Livros mais buscados
- Perfil dos leitores
- Idade dos leitores

## Ferramentas utilizadas
- SQL
- Excel
- Tableau
- ChatGPT para geração de dados

## Dashboard
Painel criado no Tableau com indicadores de:
- Total de empréstimos
- Total de livros
- Total de leitores
- Ranking de leitores
- Análise de livros, gêneros e autores mais buscados
- Quantidade mensal de empréstimos
